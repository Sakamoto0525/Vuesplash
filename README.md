# Vuesplash

## 各コンテナのビルド（初めて起動する時）
```bash
$ docker-compose build
```

## 各コンテナの起動
```bash
$ docker-compose up
```

## Vue.jsビルド・起動
```bash
$ docker-compose exec app sh
$ npm run dev
$ npm run watch
```
[localhost](localhost)で確認できる

## PHPコード整形
```
# vendor/bin/php-cs-fixer fix -v --dry-run
```
